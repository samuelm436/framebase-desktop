# framebase-app

Eine moderne C# WPF-Desktopanwendung für Windows mit folgenden Features:

- Live FPS/Frametime/1% Low Anzeige (PresentMon-Integration)
- Systeminfo (CPU, RAM, GPU)
- Pairing-Code und Upload per HTTP
- Grafikeinstellungen ändern (z.B. Fortnite-Konfig)
- FPS-Daten speichern und hochladen
- Übersichtliche, moderne GUI

## Starten

1. PresentMon.exe im gleichen Ordner ablegen
2. Projekt bauen und starten

## Hinweise
- Keine RTSS-Abhängigkeit, nur PresentMon
- Die App ist für Gamer und Benchmarking gedacht
